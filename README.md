# Customer Churn Prediction System 🚨📉

This project uses machine learning to predict customer churn in a telecom dataset.

## 🔍 What It Does

- Loads and preprocesses telecom customer data
- Trains a Random Forest model with a complete Scikit-learn pipeline
- Evaluates model performance (ROC-AUC)
- Predicts churn from new customer data
- Provides a Streamlit app for user-friendly predictions

## 📁 Project Structure

- `data/` – Raw dataset (`Telco-Customer-Churn.csv`)
- `src/` – Data preprocessing, model training, utilities
- `app/` – Streamlit app to interact with the model
- `notebooks/` – Jupyter notebooks for EDA and experimentation
- `tests/` – Unit tests

## 🚀 How to Run

1. Install requirements  
   `pip install -r requirements.txt`

2. Train model  
   `python src/model_training.py`

3. Launch app  
   `streamlit run app/streamlit_app.py`

## 🧠 Tech Used

- Python, Pandas, Scikit-learn
- Streamlit for web app
- MLflow (optional)
- Joblib for model persistence
