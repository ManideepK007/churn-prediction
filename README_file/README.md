# 📊 Customer Churn Prediction - End-to-End ML Pipeline

This project is an end-to-end machine learning pipeline to predict customer churn using structured data (Telco Customer dataset). It includes data preprocessing, model training, hyperparameter tuning, and prediction using a saved model pipeline.

---

## 🚀 Features

- Preprocessing with `Pipeline` and `ColumnTransformer`
- Random Forest Classifier with `GridSearchCV` tuning
- ROC-AUC evaluation (Best score ~0.82)
- Model saving and loading with `joblib`
- Prediction script ready (`predict.py`)
- Folder structure for scalable projects
- Ready for Streamlit or API deployment
